What is this?

Are you annoyed with navigated through files on Teams? This tool put all of your Teams folders on SharePoint, right into your desktop File Explorer.

It will:

  1. Prompt you to fetch the names of all of the Teams that you're a part of
  2. Create a folder for the General channel of every team you're a part of in Documents > Links > Teams shortcuts
  3. Set up your computer so that it connects to SharePoint every time your computer starts.
    • It does this by putting the script launch-ie.ps1 into your "$HOME\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup" directory.

How is this different from syncing a folder from SharePoint?

This is not the same an syncing. When you 'sync' a folder, it creates a copy of that folder and all of the associated files onto your local machine. Whenever you make changes, OneDrive will work to make sure that your changes are reflected on the cloud copy of those files.

With this method, you are directly accessing the files in the cloud. It's like editing the master instead of editing a copy of the master.

How do I do it?

All you need to do is click on install-teams-shortcuts.exe to run it. If that doesn't work, right-click on install-teams-shortcuts.ps1 and in the context menu, click on Run with PowerShell.

What if that doesn't work?

If that doesn't work, run the following scripts separately, in this order:

  1. teams-shortcuts.ps1
  2. launch-ie.ps1

Then, go to your $HOME\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup directory, and put the file launch-ie-ps1 in there. Anything in this folder will run on startup. This means that whenever you start your computer from a shutdown, this script will run and reconnect you to SharePoint and enables the use of your Teams shortcuts.
